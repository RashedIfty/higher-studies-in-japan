# Higher Studies in Japan — A Complete Guide to the MEXT Scholarship

An interactive web presentation for the IEEE webinar **"Higher Studies in Japan"**, covering the MEXT scholarship, application tracks, timelines, eligibility, and student life.

Presented by **Rashedul Arefin Ifty** — MEXT Scholar, University of Tsukuba.

## View the slides

Live: open `index.html` in a browser, or visit the GitHub Pages link.

## Controls

- Arrow keys (← →) to navigate
- `Esc` for slide overview
- `F` for fullscreen

## Built with

- [Reveal.js](https://revealjs.com/) for the presentation framework
- Custom CSS theme (`css/theme.css`)

## Structure

```
index.html       all slides
css/theme.css    styling
js/main.js       Reveal.js config
assets/          images
```
